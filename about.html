About SkyFly

SkyFly is a demo airline website project created to show how an online flight booking system works.

The system allows passengers to search for flights, view available options, book tickets and see a confirmation page. Admins can manage flights and bookings conceptually.

  • Fast and secure booking experience
  • Simple and clean user interface
  • Easy navigation between passenger and admin modules
